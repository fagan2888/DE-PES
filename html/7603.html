C++ Software Engineer ROV – Eindhoven
Mathematical / Physics / Algorithms / Software / C++ / Eindhoven

Introduction
We are one of the most innovative companies located in the area of Eindhoven. Currently we are building an underwater robot from scratch. We develop this ROV with camera’s, sensors and different types of monitoring systems. For the high challenging control systems on this ROV we are looking for an Engineer who has experience with C++, Python and Linux Servers.

Job Description

Our department builds state of the art Remote Operating Vehicles, we used these ROV’s underwater. The ROV’s will inspect pipelines and solid foundations.
For this ROV we need an Engineer who can program C++ regarding the control systems and camera monitoring systems. Next to this we make use of Linux Servers and the Architecture of codes, mainly core C++.
