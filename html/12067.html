Job Description : The Senior Data Warehouse Developer will be responsible for delivering performance oriented ETL and Data Warehousing solutions as part of our growing Business Intelligence organization. This position will have a broad set of database and data warehouse development skills coupled with extensive hands-on expertise using Microsoft SQL Server Integration Services. This position will also be responsible for helping DSC migrate to cloud based data warehousing solutions. This position plays a leading role in the architecture, delivery and long-term growth of an emerging and innovative analytics capability.

Responsibilities

·        Design, develop, and troubleshoot Microsoft SSIS packages that implement complex ETL processes

·        Design and develop SQL Server T-SQL objects to support data migration

·        Full development life cycle around real time integration with ETL and various data sources

·        Participate in planning and implementing the data integration and data migration activities

·        Contribute to system architectural design and planning discussions

·        Analyze, tune and optimize ETL packages, Stored Procedures & SQL scripts to ensure peak performance

·        Assist with migration and maintenance of Data warehouse in cloud

·        Data analysis on potential source systems including data profiling

·        Analyze data across multiple sources to design and document logical and physical data models, and assist with maintaining the Enterprise Data Warehouse Data Model


﻿ Qualifications

·        Bachelor’s Degree in Computer Science, Information Technology, Computer Engineering or other related field

·        At least 8 years hands-on full life cycle software development experience

·        At least 8 years of hands-on experience in SQL Server T-SQL database development (data modeling, stored procedures, functions and complex/dynamic queries)

·        At least 5 years of hands-on experience in SSIS and ETL development

·        Expert level experience at performance tuning SQL Objects (e.g. Stored Procedures, Views and Queries) required

·        Experience with CDC tables and complex DML scripts highly preferred

·        Experience with cloud technologies like AWS or Azure a plus

·        C#.net/ JavaScript experience a plus

·        Excellent verbal and written communication and organizational skills, attention to details and multi-tasking abilities

·        Strong analytical skills and problem solving skills with the ability to formulate and communicate recommendations for improvement

·        Additional project management and/or data analyst experience a plus

·        Have the ability to work effectively with others and good interpersonal skills
