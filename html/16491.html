    • Data Engineer


    Job Overview : 3 - 5 years of Experience in DATA ENGINEERING, responsible for

    building and maintaining optimized and highly available data pipelines that facilitate deeper

    analysis and reporting system. Build data processing frameworks that handle large

    volumes of business’s growing database.


    Tech Stacks:

    PySpark, Hadoop ETL development, Shell Scripting , Kafka, AWS,EC2 or any

    cloud computing exp, should have worked on product development from scratch.


    Responsibilities and Duties

    • Experience on management of Hadoop cluster with all service.

    • Ability to resolve ongoing issue with operating the cluster.

    • Create and maintain optimal data pipeline architecture.

    • Build the infrastructure required for optimal extraction, transformation, and

    loading of data from a wide variety of data sources using SQL and AWS ‘big data’ technologies.

    • Build analytics tools that utilize the data pipeline to provide actionable insights into customer acquisition, operational efficiency and other key business performance metrics.

    • Working knowledge of message queuing, stream processing, and highly scalable ‘big data’ data stores.

    • Experience with big data tools: Hadoop, HDFS, Hive, pyspark, Kafka, etc.

    • Experience with relational SQL and NoSQL databases, including Postgres and Cassandra.

    • Experience with AWS cloud services: EC2, EMR, RDS, Redshift

    • Experience with stream-processing systems: Storm, Spark-Streaming, etc.

    • Experience with object-oriented/object function scripting languages: Python, Java, etc.

    • Follow business targets and Scrum.


    Added Advantage: Apart from distributed processing using pyspark, deploying ML and computer vision algorithms using pyspark for parallel computations


    Qualifications

    • Education: Any Degree

    • Should be able to join us within 30 days of notice period

    • Other skills: Problem Solving Approach, project and Time Management skills.

    • Personal characteristics: Passion to learn and work in teams in a Startup Environment

    • Certifications: Any relevant certifications like Hortonworks/Cloudera Hadoop Certifications.


    Note:

    Candidates who are in chennai / interested to relocate to chennai and immediate joinee (less than 30days) only to apply for this job.
