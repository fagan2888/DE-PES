Job Description

Software Engineer/Analyst 2

Duties Of Position
  • Analyze the demands of work, develop and test systems in order to meet the customers demands
  • Analyze and understand the requirements of the systems/activities to be developed using the tools that we have (Label Tool, CNN, Work Request, PlanView and others) in order to develop the requester requirements;
  • Be consistent with the Label/Algorithm Development methodology for all Label request;
  • Develop the system based on the design that has been approved by the client using the languages and technologies described before;
  • Develop and perform tests on the systems, install the system in appropriate test servers to validate if the systems meet the customers needs and allow an adequate flow of information;
  • Adapt and / or adding improvements to current systems, changing systems / programs and conducting follow-up testing by the User in order to optimize the system, according to the needs of customers;
  • Propose improvements in systems performing feasibility studies, as well as monitoring and providing support in the implementation of new tools to ensure the quality and functionality of the final product.
  • Assist the project coordinator in relation to the training / service support for new developers who come to join in the customization team;
  • Keep the project coordinator always informed regarding the contacts / conversations / decisions with a client or third-party companies if the project coordinator is not present;
  • Documenting the entire process of software development in order to create a history of the project developed, thus enabling it available for future queries and updates;
  • Ensure the integrity, security and confidentiality of systems, using the standards and relevant methods of security;
  • Documenting the systems in the company's standards (new routines, improvements and other operations) to maintain a file for future reference.
  • Develop and deliver MES web report accurately and timely according to the requirement document from plant.
Qualifications Required
  • Basic language capability in English reading and writing.
  • Minimum of an IT related technical degree from an accredited institution. Bachelor's degree preferred.
  • Solid relational database knowledge, PostgreSQL preferred.
  • Intermediate knowledge on Linux and Windows.
  • At least intermediate knowledge of PLSQL, XML, Shellscript, Perl and Java.
  • Responsible, proactive, excellent team player, works well under pressure, multitasking.
