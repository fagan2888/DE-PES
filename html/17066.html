Data Engineer

TeamPlus Staffing Solution Pvt.ltd. Pune, IN

2 weeks ago

Apply on company website
Gender : Male / Female

Industry : IT Software

Basic Qualifications

2.5- 5 years of hands on experience with building productionized data ingestion and processing pipelines using any of these Java / Spark / Scala / Python / GCP / Cloud or similar.
Candidate should have an understanding of SQL concepts and experience in the same.
Good to have an experience in designing and implementing a fully operational production grade large scale data solution on Snowflake Data Warehouse .
Experience in designing and implementing production grade data warehousing solutions on large scale data technologies such as Teradata/Oracle/DB2 would be plus.
Experience of large scale Data Migration from on premise to cloud data warehouses would be plus.
Good to have a knowledge of hadoop ecosystem , hive , YARN , EMR , EC2 , S3 etc.
Experience working with Unix , Linux Operating Systems.
Excellent presentation and communication skills , both written and verbal.
Ability to problem solve and architect in an environment with unclear requirements.
Self driven with good probleming solving skills.

Preferred Skills

Bachelors degree in Computer Science/Engineering or equivalent.
Experience in building data ingestion data pipelines.
Experience in working with AWS/Azure/Google or similar data services.
Experience with dev-ops tools and shell scripting knowledge.
