Eligibility Criteria
  • Bachelor’s in Engineering in Computer Science or Information Systems
  • Experience in QuickBase / MS SQL/SAP HANA

    Primary Responsibilities:
  • Design, develop and test Data Models to import data from source systems to meet Project requirements.
  • Effectively analyze the source data and write SQL scripts to integrate data from multiple data sources.
  • Efficient in writing & analyses complex sqls
  • T-Sql programming (stored procedures, functions, etc.,)
  • Database and ETL design concepts
  • Experience in data modelling concept – Physical & Logical Data modeling
  • Experience with Data analysis, data mapping, data loading, and data validation

    Mandatory Skills required to perform the job:
  • Expert in DB design & modeling using QuickBase
  • Data Analysis skills and experience extracting information from SQL DB, SAP ECC, SharePoint, etc.
  • Strong database troubleshooting and problem solving skills are required.
  • Solid written and verbal communication skills.
  • Clear understanding of Data Warehousing concepts
  • Experience with ETL tools required
  • Strong analytical skills.

  • SAP HANA
  • Knowledge of any programming language will be added advantage (preferably .Net, JavaScript)
  • Understanding and Experience in Manufacturing, Supply Chain, Spares Operations domain
