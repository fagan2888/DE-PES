Primary Job Purpose :


  • Develop methodologies and analyze manufacturing data from multiple domains.
  • Develop and implement user interface tools for data users
  • Facilitate quality improvement to the product and process by providing customized application tools
  • Develop and manage data base applications


Qualifications Required:

  • Degree in Information Technology/Compute Engineering/Electrical / Electronic / Mechanical
  • Strong software development experience is a must:
  • Data analytics
  • User interface application
  • Data base management
  • Data mining
  • Well verse in SQL, Visual Basic, Visual C#, Java etc.
  • Able to learn and develop software in different platform independently.
  • Working experience in semiconductor, automotive electronics, manufacturing environment is preferred.
  • Knowledge in application of quality engineering tools, Six sigma, quality system implementation (TS 16949/QS 9000/ ISO 9000), reliability or failure analysis will be an added advantage.
  • Good interpersonal, leadership, analytical and communication skills.


Interested candidate please send your updated resume in Words Format , stated your current/expected salary and email to hr@envirodynamics.com.sg
