Front End Developer


As a Front End Developer, you will create high performant UI for web, desktop, mobile applications, by writing readable, testable, and maintainable code.


Your Key Responsibilities

Design, code, and test based on user stories and other system engineering artifacts as part of an agile scrum team;

Ensure that code is modifiable, efficient, reliable understandable, and fault tolerant;

Support all phases of the development lifecycle, including requirements analysis, system design, development, testing, documentation, and maintenance;

Collaborate with Backend Developers, Designers, and QA.


Desired Qualifications: [Essential]

3-6+ years of professional software development experience;

2 years of React using the newest technologies such as React Hooks and Redux;

2 years of Web Development experience with HTML5, CSS, JQuery, Javascript;

Knowledge of VUE.JS

Experience in prototyping, wireframes etc.;

Familiarity with other JavaScript frameworks;

Familiarity with CI/CD;

CSS, SASS, Responsive Web Design, Single Page Application development;


A plus if you also have: [Desirable]


Knowledge of Java, JEE, Spring, Hibernate;

Knowledge of relational databases;

Knowledge of Cloud Technologies – AWS.
