    • Are you a passionate developer with over 4 years of professional experience that wants to work remotely with a US-based startup? okapi is a marketing platform that helps companies like Anheuser-Busch, Reign Energy Drink, SparkingICE and Maui Jim Sunglasses collect data, improve operational efficiencies, and make smarter decisions with their experiential marketing budgets. We're a US-based software startup and looking to expand our team in Romania.


    Qualification and Requirements:

    • Minimum four years’ experience developing software.
    • At least one hour of overlap with US Central Time Zone
    • Advanced understanding and experience with the following technologies required:
    1. Relational SQL Databases (PostgreSQL)
    2. JSON and JSONB data strctures
    3. Javascript (React/Node.js)
    4. HTML5/CSS3
    5. Cloud Infrastructure (AWS)
    6. Web Server Configuration (Nginx/Express)
    • Experience with the following technologies a plus:
    1. GRAPE stack
    2. GraphQL (Apollo)
    3. React UI frameworks (Ant Design)
    4. Docker and containerization
    5. Cerbot/Let’s Encrypt
    6. Amazon Web Services (ACM, CloudFront, EC2, RDS, Route 53, S3)
    • Ability to work with a variety of development tools including Bitbucket, GitHub, Jira, Atom, and Visual Studio
    • Excellent written and verbal communication skills.
    • General understanding of the full software development lifecycle.
    • Familiarity with continuous integration principles preferred.
    • Experience with test driven development a plus.


    Key Responsibilities

    • With general supervision, develops, documents, maintains and unit tests code by utilizing software development environments and programming languages.
    • Develops, documents and maintains software design by applying established methods and practices. Performs troubleshooting of defects.
    • Collaborates with others to propose solutions to problems.
    • Gather and organize data for use in plans, schedules and estimates.
    • Assist in the specification and maintenance of software requirements.
    • Works with the rest of the development team in a self-organizing SCRUM approach to software development including:
    • Daily standup participation and communication of status.
    • Demonstrates progress and end-of-sprint demos.
    • Participates in post-sprint retrospectives with an honest and constructive approach.
    • Integrates and compiles software components into builds and actively participates in production operations duties as assigned.
    • Implements software projects within estimated hours while meeting or exceeding okapi quality standards and client expectations.
    • Performs problem identification and provides fixes for support and operational issues.
    • Interfaces with the entire okapi team, including various internal departments and all levels of management.
    • Interfaces with clients in a Level 2 Support role.
    • Performs knowledge transfer to Client Success and Support team members.
    • Participates in product development efforts to broaden the core service offerings of okapi.
    • Develops and maintains current technical and professional knowledge-base by attending industry-related user group meetings, vendor trainings, conferences, and seminars.


    Description

    The Full Stack Developer II (Dev II) provides software development and support for the okapi platform. Assists with the documentation and maintenance of architectures, requirements, algorithms, interfaces and designs for software systems. Develops and maintains code and integrates software components into a fully functional software system. Assists with test procedures to ensure software platform requirements and functionality are met. Gathers information to support software project management. Documents deployed processes and tools and collect metrics. Supports software research and development projects. Troubleshoots software issues. Follows company software standards and processes. Works in an agile software development environment. Reports to the Product Owner.
