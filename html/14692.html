    Job Description

    As a Data Science Engineering-Intern in Micron, you will work with data science engineers and other domain customers to help the team for moving forward projects. We are looking for bright self-motivated individuals who are driven to solve real world data problems, both big-data and small, using various tools that are widely available in data science. Your main responsibilities include, 1) Build Jupyter notebooks that ingest and process data at scale and in real-time 2) Program the business logics using Spark, Scala, Python, and/or Java 3) Help end user to set up Jupyter notebook environment to run scripts 4) Provide data solutions to improve manufacturing cost, cycle time, output and quality

    Education background

    Bachelors/Master degree in Computer science, Software Engineering, Electrical Engineering, or related fields.
