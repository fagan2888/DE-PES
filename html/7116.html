    ¿QUÉ TIPO DE PERFILES BUSCAMOS?


    -Diseño e implementación de procesos de carga (ETL, ELT).


    -Servicios de data en Azure:

    o Azure Data Factory v2.

    o Apache Spark (eg. Azure Databricks).

    o Azure Data Explorer.


    -Lenguajes de programación:

    o Scala.

    o SQL.

    o .NET.


    -Devops en Azure:

    o ARM.

    o PowerShell / AZ CLI .


    -GIT, Graddle / SBT / Maven, Azure Devops, Jenkins
