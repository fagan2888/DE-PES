Job Description

Are you experienced enough to work within a team which is responsible for supporting MS SQL databases for the customers? We are currently looking for experienced engineers. Apply now.

Responsibilities
  • Daily jobs execution monitoring,
  • Capacity planning and monitoring on storage,
  • Capacity planning and monitoring (DB and Backups),
  • Providing support on major incidents (Open Hours),
  • Liaising with local teams for regular reboots, maintenances, and infrastructure hosting topics,
  • Reviewing and maintaining DB Backup and maintenance jobs (indexes, statistics),
  • SQL performance analysis and improvement,
  • Monitoring and troubleshooting of the liberty reporting spooler,
  • Performance monitoring and improvements in SQL jobs,
  • Monitoring backup and maintenance jobs execution – solving errors,
  • Reviewing and improving the DB maintenance plan,

Requirements

You should have;
  • Knowledge of Microsoft server operating systems,
  • Management of Microsoft SQL Server databases,
  • Creating copies and backups of databases for development and test purposes,
  • Reporting services,
  • Integration services.
