Job Description

ML and BI are two vital areas for the continued growth of Mercari’s services. As a data engineer, you will work on the design, development, and operation of the next-generation data infrastructures and data pipelines needed to utilize data in these two areas.

  • Development and operation of streaming data pipelines that use Google Cloud Platform’s (GCP) Cloud Pub/Sub, Dataflow, GCS, BigQuery, Stackdriver etc.
  • Development and operation of batch data pipelines that use GCP’s Cloud Dataflow, GCS, BigQuery, Cloud Composer, etc

業務内容

メルカリのサービスの成長の鍵となる、MLやBI領域でのデータ活用を実現するための次世代データインフラストラクチャ、データパイプラインの設計・開発ならびに運用を行います。

  • Google Cloud Platform(GCP)のCloud Pub/Sub, Dataflow, GCS, BigQuery, Stackdriver等を利用した、Streamデータパイプラインの開発と運用
  • GCPのCloud Dataflow, GCS, BigQuery, Cloud Composer等を利用した、Batchデータパイプラインの開発と運用


REQUIREMENTS

Minimum Requirements

  • Mercari's mission and values resonate with you
  • At least one of the following:
    • Experience developing high-volume stream data processing systems using distributed processing frameworks (Dataflow/Spark/Storm/Kafka/Flink, etc.)
    • Experience developing high-throughput batch systems using workflow engines (Airflow, Luigi, Digdag, Azkaban, etc)
  • A degree in computer science or a related field and over 5 years of experience with software development
  • Ability to discover and solve technical system issues (such as a drop in performance)
  • Experience developing software using Scala or Java, and Python

Preferred

  • Experience with high-traffic data processing using GCP or AWS
  • Experience with developing applications using Apache Beam, Spotify Scio
  • Knowledge of and experience with metadata management and web service logging
  • Knowledge of data schemas represented by Avro and Protocol Buffers, along with file formats (Parquet/ORC, etc.)
  • Understanding of the collection and practical use of data gathered from logs when using microservice architecture
