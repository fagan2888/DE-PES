Full Stack Java Senior

Desideriamo entrare in contatto con esperti Full Stack con minimo 7 anni di esperienza su progetti di netto spessore.

E' richiesta una solida conoscenza dei principi e metodi di sviluppo Web e/o di backend:

  • modello Object Oriented
  • modello DB relazionale e SQL
  • Versioning Git o Mercurial o SVN
  • Clean Code

Necessaria la conoscenza di:

  • Architettura Java EE
  • Java Web Applications (Servlet API, Servlet Containers, Apache Tomcat)
  • Spring framework
  • SQL, JDBC, Spring JDBC
  • Servizi REST con Spring MVC o JAX-RS
  • preferibilmente anche JAX-WS
  • Maven
  • Junit

Sede di lavoro: Genova
