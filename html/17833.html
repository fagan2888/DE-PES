IT Cloud operations analyst

Wolters Kluwer Lucca, IT

4 giorni fa

Candidati sul sito Web dell’azienda
The IT Cloud Operations Analyst will be responsible for the maintenance, monitoring, diagnostics, support, and installation of systems on Cloud datacenters (Azure, AWS) for the supply of CCH Tagetik Cloud Service

The ideal candidate would be innovative and have great problem solving and analytical skills.
You have experience operating distributed systems and diagnosing and resolving complex problems.

You pay attention to detail and solve problems at their root.

Key Duties And Responsibilities
  • Install new/rebuild existing servers and configure hardware, services, storage, etc. in accordance with standards and project/operational requirements
  • Set up and configuration of common infrastructure components (Networking, VPN tunnel, Firewalls, DC, etc..),
  • Manage Microsoft Windows, Linux systems, Commercial and Open Source Databases (PostgreSQL, Sql Server, Oracle, SAP Hana)
  • Work closely with service desk and other colleagues to resolve problems
  • Being able to communicate effectively with the customer
  • Use Monitoring tools to get insights and proactively resolve problems before service disruption
  • Investigate and troubleshoot on system and software configuration and related performance issues
  • Apply OS/application patches and upgrades on a regular basis and upgrade administrative tools and utilities
  • Being able to work on new software and services configuring and documenting as necessary
  • Research and recommend innovative, and where possible, automated approaches for system administration tasks

Desired Skills
  • Bachelor's Degree in Computer science or related experience in technical field
  • Experience in IT server administration, in particular with Linux systems
  • Good understanding of TCP/IP networking
  • Microsoft Active Directory and Group Policy Management
  • Enthusiasm and passion for work
  • Flexibility, autonomy and proactivity
  • Attitude to work in team
  • Customer centric attitude

Preferred Skills
  • Familiarity with Cloud infrastructure (AWS/Azure)
  • Proficiency in working with Bash and PowerShell
  • Experience with Monitoring Solutions (Nagios, Grafana, Kibana, AWS/Azure monitoring tools…).
  • SAML Federation
  • ITIL
  • Strong Problem solving and critical thinking skills.
  • Fluency in English
  • Experience working in DevOps environment.
  • Container Technology (e.g. Docker).
  • Infrastructure as Code experience (e.g. Chef, Puppet, Ansible, etc).
  • Strong awareness on security matters
  • Familiarity with operations in ISO27001 certified context

What do we offer you?
  • Continuous learning and development programs
  • A team of passionate technologists, using state-of-the-art technology stack and collaboration models
  • A fast-moving environment that values "getting things done" while also steadily increasing process maturity
  • A wealth of challenging projects, and a team that will support you in mastering them
