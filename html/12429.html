Java Software Engineer

Croatia osiguranje d.d. Zagreb, Croatia

2 weeks ago

Apply on company website

Java Softverski Inženjeri u Croatia osiguranju rade na razvoju modernih poslovnih aplikacija osiguravajućeg društva, razvoju novih funkcionalnosti i integraciji s vanjskim sustavima.


Odgovornosti :

  • Osmišljavanje, projektiranje, programiranje i testiranje novih i postojećih aplikativnih rješenja u skladu sa razvojnim standardima Društva
  • Analiziranje zahtjeva korisnika kroz suradnju s poslovnim analitičarom te predlaganje optimalnog rješenja
  • Analiziranje protoka podataka, korištenja podataka i poslovnih procesa
  • Utvrđivanje operativne izvedivosti evaluacijom i definicijom problema
  • Usavršavanje razvojnog procesa predlaganjem novih ideja i koncepata
  • Praćenje novih trendova i tehnologija
  • Analiziranje prilagodbe postojećeg sustava novim tehnologijama
  • Proaktivnost i kreativnost u rješavanju zahtjeva i problema
  • Kolaboracija s timovima prilikom kreiranja idejnih rješenja i rješavanja problema
  • Mentoriranje pripravnika


Uvjeti :

  • 2 godine iskustva u Java programiranju koristeći Java Enterprise Edition ili Spring Boot Framework
  • Iskustvo u barem jednom od navedenih aplikacijskih servera: Tomcat, Weblogic, Websphere, JBoss
  • Iskustvo u radu s relacijskim bazama podataka
  • Korištenje JUnit frameworka za razvojno testiranje
  • Razumijevanje REST principa i ograničenja REST arhitekture, kao i dizajniranje RESTful API-ja
  • Korištenje CI/CD alata
  • Iskustvo u radu sa jednim od navedenih build alata: Maven, Gradle
  • Korištenje jednog od navedenih sustava za verzioniranje koda: GIT, SVN


Prednosti:

  • Iskustvo u radu sa Oracle bazom podataka
  • Iskustvo u redu sa Jenkins CI/CD alatom
  • Korištenje Dockera i Kubernetes orkestracije
  • Razumijevanje mikroservisne arhitekture sustava
  • Iskustvo u razvoju u financijskom ili osiguravateljskom sektoru
  • Razumijevanje servisno orijentirane arhitekture
  • Rad unutar SCRUM procesnog frameworka
