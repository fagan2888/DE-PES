SYSTEM TEST ENGINEER

Valeo is a leading supplier of components and systems for the world's automakers. In our R&D (Research & Development) Center in Prague we develop advanced driver assistance systems (ADAS) for intelligent cars.

We are looking to our team a skilled System Test Engineer, who will participate on creation of test guidelinesand data analyses from the testing process

Job Overview

As a System Test Engineer you will be responsible for:
  • Specification, planning, execution and analysis of system tests
  • Creation of test strategy
  • Do the link between system specification and system test specification
  • Initial system start up including integration tests
  • Development of system analysis and system testing tools
  • Problem analyses, internally and externally
  • Presentation of the system tests for internal and external customers
  • Assisting the development of test systems for vehicle testing
  • Creation of application instructions
  • Performing the system tests
  • Organizing of the system tests
  • Collaboration on the specification of system requirements, requirements analysis (marketing, customer, internal and customer projects)
  • Collaboration on derivation of requirements for mechanics, hardware and software in close cooperation with the departments
WE REQUIRE

Essential Requirements Are
  • Degree in Software Engineering, Computer Science, Electronic Engineering or equivalent
  • Experience with requirements tools and design tools
  • Understanding of software development process
  • Excellent English communication skills, both written and verbal
  • Driving licence B
Highly beneficial would be if you have any knowledge or experience with:
  • CANOE, Canalyzer, Labview
  • System testing and automotive is an advantage
  • Programming in C, C++ on μC or DSP
  • Using German language
Personality Preferences
  • Quality oriented engineering attitude,
  • High level of motivation and work ethic with an emphasis on practical, hands-on work
  • High degree of innovation and creativity
WE OFFER
  • Work in a stable international company in a multinational team
  • Work with cutting-edge technologies in Automotive
  • Cooperation with other development sites
  • Possibility to see your work product in serially produced cars
  • Possibility to see the whole development process in one place - from an idea (customer requirement) to a product (e.g. automated parking ECU)
  • Developed algorithm validation on a company test track near Prague
  • Personal development - Special Trainings and Language Courses
  • Package of company benefits (Bonus Structure, Pension scheme, Meal wouchers, Flexi Passes, 6 Weeks of Holidays)
