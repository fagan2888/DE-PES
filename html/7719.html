Data Scientist

Gauteng


Companies on OfferZen are hiring for data scientists to:

  • Build machine learning models to make predictions and answer key business questions

  • Develop data models and products to produce insights

  • Perform feature engineering, visualisations and other techniques to make out certain perceptions from the available data

  • Use statistics tools to analyze data to understand and model the relationship between different metrics and identify key insights

  • Collaborate with teams to identify opportunities to build data science products


Requirements

  • 2+ years experience working as a data scientist

  • Honours/Masters/PhD in Computer Science, Mathematics and Statistics or related technical discipline (or equivalent working experience)


Skills

  • Expert level knowledge in SQL

  • Experience in ETL tools and programming languages like Python, R or Java

  • Strong experience in statistical insights and machine learning - Mahout, Bayesian, Clustering etc.

  • Working knowledge of database systems

  • Proficient in mathematics, statistics, correlation, data mining and predictive analysis
