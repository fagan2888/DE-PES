Informácie o pracovnej ponuke

ID: OPSK - 1123

Lokalita: Slovenská Republika, Košice

Ponúkaný plat: 1400€

Popis pracovnej činnosti, povinnosti

Hľadáme nových členov nášho tímu, ktorý pracujú na projekte pre celosvetovo známu výrobnú spoločnosť. Budú členom novo založeného tímu Big Data ako hlavný vodič existujúceho dátového centra IoT. Tento tím sa zúčastňuje projektu nemeckého ústredia. Aktuálne otvorená pozícia Data Engineer pre atraktívnu softvérovú IT spoločnosť v Košiciach. Spoločnosť sa zaoberá vývojom a implemantáciou softvéru SAP riešení, nezávislé testovanie a zabezpečovanie kvality IT služieb v rámci energetického sektora.

Benefity

Úspešný kandidát bude pracovať na migrácii existujúcej systémovej krajiny (Java EE) do architektúry mikroprocesov založenej na platforme Spring Boot na platforme Pivotal Cloud Foundry (PCF). Pripojíte sa k nášmu tímu vývojárov a testerov so sídlom v Košiciach a môžete prispieť k pravidelnému vývoju funkcií, ako aj k architektonickým zmenám, vylepšeniam a vývoju systému.

Požiadavky

Ďalšie zručnosti
  • Niekoľko rokov relevantných skúseností v oblasti dátového inžinierstva, správy údajov alebo vývoja softvéru
  • Dobré a preukázané programovacie zručnosti, napr. Python, Java, C ++
  • Relevantné skúsenosti s modelovaním údajov (ER, UML), návrhom databázy a administráciou databázy
  • Rozšírené znalosti o relačnej a nerelačnej architektúre databázy, napr. Microsoft SQL Server, Oracle, MongoDB atď.
  • Skúsenosti s cloudovými princípmi, - životné prostredie, --architektúra a - technológia
  • Praktické skúsenosti s integráciou zdrojov údajov (ERP, IT systémy atď.) S potrubiami ETL alebo virtualizáciou údajov
  • Praktické skúsenosti s cloudom Azure Cloud, ako je Azure Data Factory, Azure Data Lake Storage, Azure Event Hub, Azure IoT Hub a Databricks.
  • Prvé skúsenosti s veľkými dátovými nástrojmi ako Hadoop, Spark, Kafka, Hive atď.
  • Skúsenosti s nástrojmi na virtualizáciu údajov, ako je Denodo
