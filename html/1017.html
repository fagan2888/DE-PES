Job Description

Looking for an amazing opportunity to work alongside a great and dynamic team? Then look no further because we have the perfect position for you, all you need is to be ambitious, enthusiastic, hardworking and to meet the below requirements. 

Key Requirements
  • 3 – 5 Years’ experience
  • Experience with implementing/utilising multithreaded data processing
  • Fluent with both relational algebra and MapReduce logic
  • Git version control
  • Experience with open source big data infrastructure: Hadoop, Spark, Mongo dB, Hive, HBase
  • Comfortable with Linux command line
  • Experience with SSIS, SAP Data Services, IBM, Pentaho or equivalent
  • Ability to integrate large data files on the fly
  • Experience with BI tools advantageous
  • Must be detailed, orientated and able to handle multiple tasks at one time
  • Excellent written and verbal communications
