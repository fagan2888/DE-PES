PURPOSE AND OBJECTIVES



    • Services&SupportTechnology (SST) is a part of global DigitalBusinessServices (DBS) organization in SAP. In SST each team frequently collaborate, take responsibility for global cloud application development, product quality management, enterprise standard compliance, continuously delivery and cloud native operation. By closely cooperate with each project teams, we help DBS continuously fast innovate in cloud application architecture, delivery, services transformation, continuously improve customer experience by offering highly automatic predictive supports with cloud services applications running on SAP CloudPlatform (SCP) and Hypterscalers.

    QOD as an important team for transformation and innovation, we practice DevOps culture, supporting development teams by providing CloudFoundry & Kubernetes Cloud platform based on various cloud infrastructures, leverage application quality and secure level, continuously deliver innovations via automatic pipeline. We are the front runner in creating pipeline, automatic cloud application monitoring, scaling, operation tools.

    We are looking for research oriented and innovative engineers to join this fast-growing team, to focus cloud technic area, by developing & operating different automations, defining InfrastructureAsCode, support DBS ML/AI infrastructure, establishing the foundations for DBS DataLake. Some roles will also cross-region support global cloud application operations to ensure commercialized SaaS functional and non-functional requirements like application performance, resilience, scalability, usability, security, and corporate compliance are satisfied our customers.

    EXPECTATIONS AND TASKS



    • The Major Objectives For QOD Team Are
    • Drive the improvements, incubate innovations, create automations for DEVOPS culture from processes, methodologies, architectures, technology aspects;
    • Cross-region cooperate with global operation teams to support DBS cloud native applications together;
    • Contribute to DBS innovation projects like DwC, DataLake and AI by a self-driven, self-managed way.

      The position needs to process one or several of the below tasks : (see the marks and next section for the skill requirements)
    • Operation and Monitoring DBS SaaS in productive, to troubleshoot issues in complex, distributed, multi-tier architectures. Support operation teams in other regions.
    • Implement SAP/OpenSource tools(1), or create automation tools by your own(2) to fit the operation need.
    • Create and maintain ContinuousDelivery/ContinuousDeployment (CD) pipelines for Cloud applications. (3)
    • Integrate TestAutomation(4), SecurityScan(5), SAPComplianceProcess into the pipelines, and provide suggestions to dev team when scans in failure; (6)
    • Join local scrum meetings as a QualityEngineer, drive high coverage of UT and create Integration/E2E test automations (7)
    • Work closely with cloud application development scrum teams, clearly communicate the status and tasks; (8)
    • Manage IaaS accounts from SAP and Hypterscalers, provide PaaS environment to DBS development teams (9)

    Skill, Competencies/Experience



    • To full fill the tasks and challenges, we are seeking candidates who has related experience, ~3 years for Engineer position, ~6 years for Senior position, ~10 years for Expert position

    Depends on the different task requirements, you need to familiar with one or some of below knowledge and experience:
    • e.g: Kubernetes, Knative, Istio, Prometheus, Dynatrace, etc
    • Programming language is not limited when creating cloud based automation tools, candidate need to well skilled to use one or some of the programming language(s) like: Python3, Java, Golang…
    • Groovy, Linux Shell
    • Integrate tools like Selenium, Robot Framework
    • Fortify, Checkmax, and other security scan tools;
    • Need security knowledge in networking, frontend UI authentication, digital signature & certificates, JWT, software threat modeling etc;
    • scripting for test automation tools e.g: Selenium, OPA, SUPA, START
    • knowledge of software development lifecycle, scrum, Agile software development. Exposure to large scale software build, integration, and test environments
    • Knowledge and experience of Hyperscaler cloud AWS, Azure, GCP, AliCloud IaaS and their major provided services;

    Education, Qualifications



    • BS or MS degree in related technical or science fields, e.g: software engineering/computer science/communication/mathematics/physics/etc…

    Is a self-starter, quick learner; capable of setting priorities and remain efficient while multi-tasking.

    Good communication skills in English (both written and oral) across multiple teams is a required.

    Clear mind, ability to break down complexity and communicate things in a clear manner.

    Team work /independent work and manage time effectively.
