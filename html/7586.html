Senior Frontend Developer

Deegloo Zagreb, Croatia

1 week ago

Apply on company website

Are you ready to move your Angular and React skills up to the next level by joining us as a Senior Frontend Developer? If you want to come work in a pleasant environment and build challenging projects from scratch, hurry up and apply now 😊


We create web-based solutions with beautiful user interfaces and user experiences taking care that all our work meets high-quality standards. We’re mostly young people and our clients come from the USA Dairy Industry.



Who we’re looking for


Senior Frontend Developer who is passionate about Angular and/or React (excellent knowledge of one of these is sufficient). You’ll work in the international team where development work is done following agile methodology.


Technology stack:

  • Angular or React
  • TypeScript
  • webpack, npm, yarn
  • Amazon AWS
  • Git



What you’ll be doing


You’ll be involved in building frontend for our projects and participate in a 5-6 people Scrum team. Your role will be to communicate with backend developers and QA, consume REST endpoints and implement user interfaces as defined in the specifications and design mockups.



We can offer you


  • An opportunity to build platforms from scratch and influence architectural decisions
  • Opportunity to take the role of a Team Leader
  • MacBook Pro and a 4k monitor, a plant aside if you prefer green
  • Flexible working hours and work from home up to six days per month
  • High class in our new office built from scratch for us
  • A compensation package based on the value you provide
  • An educational budget for any books you would like to read, conferences you want to participate in, or certificates you would like to apply for
  • Cross-functional Scrum teams of 5-6 people participating in agile development
  • Code reviews



Do you like what you’ve read? If you did, apply now!


P.S. We’re processing applications as they arrive, so hurry up! ;)


Application deadline: 28.10.2019.
