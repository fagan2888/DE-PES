Job DescriptionSenior Data Engineer ( Python / AWS / SQL ) London 6 Months £550 - £650 per day Senior Data Engineer ( Python / AWS / SQL ) is urgently required to join one of my market leading FinTech clients in London on a contract basis. You will be the link between the Data and Back end teams to improve data efficiency throughout the organisation. This is an exciting opportunity for a Senior Data Engineer ( Python / AWS / SQL ) to utilise your Data Engineering skills to implement optimal solutions and strive for greater functionality in the data systems. Key responsibilities for a Senior Data Engineer ( Python / AWS / SQL ) include:
  • Play a key part in the creation of a scalable data platform tools and solutions which are used for processing high volumes of data.
  • You will play a key part in a large-scale Data ingestion project
  • Utilising Docker to create, deploy, and run applications by using containers
  • Taking care of software provisioning, configuration management, and application deployment.
  • Provide end-to-end engineering and architecture services. Essential skills for a Senior Data Engineer ( Python / AWS / SQL ) include:
  • Expert skills in Python
  • Strong commercial experience with AWS
  • Highly experienced in SQL coding
  • Worked extensively on SQL and a variety of database technologies.
  • Strong experience building data pipelines
  • Ability to hit the ground running and portray an eagerness to learn.
  • Strong interpersonal skills and the ability to communicate with all levels in the organisation. Desirable skills for a Senior Data Engineer ( Python / AWS / SQL ) include:
  • Experience of Neptune
  • Highly skilled in Big Data technologies (Spark, Kafka)
  • Good understanding of Docker and containerisation
  • Experience of Jenkins, Airflow & Team City A demonstrable track-record in a similar role is a necessity for this business-critical role. The culture is one that promotes creativity and the ability to think outside of the box. This is an exciting opportunity for a Senior Data Engineer ( Python / AWS / SQL ) which will allow the individual to work with the industries most advanced technologies.
