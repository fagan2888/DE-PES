Description

Develop software applications and software integration solutions utilizing JAVA, Spring(Boot) framework, CI/CD. “Information Systems Engineer” position is a combination of Software Systems Integration, Software Development, DevOps and database system knowledge, Release Management, Configuration Management and Continuous Code Quality.

Job Description

  • Design and develop software solution, deploy and troubleshoot problems in active QA and Dev environments, along with development and deployment infrastructure. Manage configuration and patching. Recommend system-wide reliability improvements.
  • Help Software Deployment engineers with deploy scripts, configuration, issue investigation and SL3 support function.
  • Design, development, debugging and support of corporate software applications using Java or other high-level or script programming languages according to specific coding standards.
  • Research & implementation of algorithm problem solutions.
  • Accurate and precise code documenting
  • Creation of high-level functional specifications and design documents.
  • Design and working out tests for quality assurance of implemented software applications
  • Assistance SQA team to ensure the software is of high quality
  • Participation in all company’s obligatory trainings and meetings.
  • Participation, if necessary, in on-site work aimed at realization, debugging and implementation of software applications.
  • Additional responsibilities: Other duties may be assigned

    • Requirements
    • Applicant must have strong Java experience and fluency in object-oriented design techniques.
    • Bachelor in computer science or software engineering or equivalent degree with minimum of five years of industry experience
    • Applicant must have strong experience with relational database management systems
    • Applicant must have experience of building data processing pipelines and knowledge of distributed systems.

    Education Bachelor’s Degree in Computer Science, or equivalent
