Data Software Engineer

V-IT nv Brussels, BE

6 days ago

V-IT is looking for a Data Software Engineer for our client in Brussels.

Tasks
  • Participate to the definition of the needs and writing of the User Stories
  • Collaborate with Data Scientist to the development of the Advanced Analytics modules
  • Participate to the Data Architecture and modelling on an Hadoop platform
  • Develop modules related to the Data ingestion, Data modelling and Data aggregation
  • Ensure quality of the development using Continuous Integration framework
Skills
  • At least 2-3 years as a Data SW Engineer
  • Experience in SW development on an Hadoop environment
  • Hadoop ecosystem
  • Advanced Analytics framework and libraries: PySpark, H2O, Python Mlib
  • Scala and Python
  • Continuous Integration and Deployment framework: Gitlab, Jenkins, Ansible…
  • Data Visualisation framework like QlikSense is a plus
  • English
