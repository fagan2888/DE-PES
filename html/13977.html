Position Title: Data Scientist

Core Responsibilities
  • Analyze big data of ad optimization system, working closely with the product team
  • Constantly experiment and improve the quality of existing machine learning models for the recommendation/ad optimization system
  • Invent and analyze big data for given business cases and help implement them in production
  • Work with the product/engineering team to implement, test, and deploy the solution on Google Cloud Platform


Core Competencies
  • Passionate about machine learning and big data
  • Ability to communicate complex algorithmic works as easy-to-understand story telling
  • Motivated to test at billions scaled data and follow up machine learning models in product


Requirements
  • B.S./M.S., in Computer Science or Statistics or similar applied technical field
  • Solid programming skill of scripting language Python
  • Ability to work in a fast paced, test-driven collaborative and iterative programming environment
  • Understanding of Algorithms, Data Structures and Machine Learning / Data Mining
  • Understanding of RDBMS, SQL and NoSQL alternatives
  • Experience with Hadoop, Hbase, Pig or PySpark is a plus
