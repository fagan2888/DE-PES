    Recruitment Consultant - DevOps at Revolution Technology hollie.sharp@revolutiontechnology.co.uk

    • Site Reliability/ DevOps Engineer (GCP, Google Cloud Platform, AWS, IaC) – London – £75,000 - £80,000

    Our client is a global data and measurement-driven media agency whose mission is to make advertising more valuable to the world. They are looking for a Site Reliability/DevOps Engineer to join their team to build a culture of continuous improvement of our processes, infrastructure, and practices around the infrastructure and operations.

    Site Reliability/ DevOps Engineer (GCP, Google Cloud Platform, AWS, IaC) will need:

    • Site reliability engineering principles
    • Expertise in Infrastructure as Code and technologies that support it
    • Hands-on approach to the implementation of the infrastructure as code (IaC).
    • Strong knowledge of Kubernetes and Docker
    • Experience working with Jinja2 and Gitlab CI
    • Google Cloud Platform and AWS
    • Programming in Python - you know how to read code; you can understand it and you can make changes
    • Comfortable in Bash
    • Knowledge of networking and cloud-based delivery models
