Job Description

At LeoVegas, data is essential for all our decision making needs for example customer behaviour, marketing activities, social media, product development and discovery of new markets. This data is deeply valuable and gives us insights into how we can continue improving our service for our customers and partners. Our data team is seeking a hardworking data engineer with a strong technical background and who is excited about developing state of the art analytical solutions using the latest Big Data technologies.

Our data infrastructure is built using Google Cloud Platform (GCP). The focal point is our BigQuery data warehouse where all data from various sources is centralised for analysis and reporting. We process data in both batch and streaming mode.

Our Main GCP Tools Are
  • Dataflow/Apache Beam
  • BigQuery
  • Pub/Sub
  • Composer/Apache Airflow
We use code for all components to enable test automation and version control. There is a strong focus on data quality and automated testing. You will need to have a mindset of writing clean and maintainable code, this since it is the right thing to do, but also since we are a small team and therefore can't afford a big backlog of bugs and maintenance.

Remote Location is also an opportunity.

Qualifications
  • Working Experience with ETL, building pipelines using Big Data processing techniques
  • Proven Knowledge of Java SE and SQL.
  • Experience in Data warehousing
  • Working Experience with Automated testing
  • Experience with Python
