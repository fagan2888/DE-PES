    Executive Assistant / HR Officer

    • LOGOS IT Services is looking for a JAVA Software Engineer to join our teams.


    • Job Purpose


    The JAVA Software Engineer is responsible for building high-performing, scalable, enterprise-grade JAVA / J2EE / Spring Boot software applications while providing expertise in the full software development lifecycle, from concept and design to testing.

    Confirmed Software engineer will join Development teams on customer’s projects or assigned on internal projects, both assignments for the financial Sector.


    • Duties, Tasks and Responsibilities


    The JAVA / J2EE Software Engineer performs the following activities:


    1. Write functional and technical documentation

    2. Design and develop high-volume, low-latency applications for mission-critical systems, delivering high-availability and performance

    3. Contribute in all phases of the development lifecycle

    4. Write well designed, testable, efficient code

    5. Ensure designs are compliant with specifications

    6. Prepare and produce releases of software components

    7. Support continuous improvement by investigating alternatives and technologies and presenting these for architectural review.

    8. Report to a Team Leader

    9. Provide support for Business IT queries and technical support to younger colleagues


    • Qualifications


    Education:

    • Bachelor or Master degree in Computer Science, IT Engineering School

    • 3/5 years’ experience in significant projects


    Knowledge, skills and abilities:

    • Good Java / J2EE / Spring Boot programming skills;

    • Profound insight of Java and JEE internals (Classloading, Memory Management, Transaction management etc);

    • Usage of Java technologies (EJB, JMS/MDB, JSF, JSP, framework Hibernate) ;

    • Usage of JavaEE Application Servers (ideally Redhat EAP Jboss or Oracle Weblogic);

    • Oracle database, SQL and PL-SQL;

    • Working with Linux environment;

    • Experience with Eclipse, Maven, Jenkins, Git and SVN Environments;

    • Object Oriented analysis and design using common design patterns;

    • Team player with good communications skills, ability to identify problems and propose solutions;

    • Fluent or Good Command in English and French, German is a plus;

    • Knowledge in Financial sector is a real asset (SWIFT, Capital Markets…);

    • Able to ramp-up with a reduced timeframe learning curve

    If you want to join a dynamic company and strong of his experience for more than 20 years, send us your CV to the address info@logos-its.com . We will get back to you quickly.
