About Gameloft

Job Description

Embedded into an analytics team, this role will collaborate closely with Big Data Engineers and
Data Analysts building different analytics solutions for the company. This profile is fully focused
in shaping the data so it’s structured, clean, usable and understandable for our clients the
game teams.

Responsibilities
  • Define Data Model
  • Create and maintain ETL processes
  • Define and create BI tool measures, dimensions and dashboards that are globally used across games
  • Give support to analysts

Qualifications

Hard skills
  • +3 years of proven track record as a Data Engineer
  • Proficiency in at least one common query language
  • Proficiency in Data modelling
  • Knowledge in ETL processes and tools
  • Worked with Python and Spark
  • (Nice to have) Worked in large scales systems providing huge amounts of data

Soft Skills
  • Ability to understand the business needs
  • Ability to tailor technical solution to user and client requirements
  • Ability to provide simple output for client
  • Interest for the gaming industry
