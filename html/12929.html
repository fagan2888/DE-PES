Project Description

The FRP (firmwide reporting platform) migration project is a key delivery for 2019 which focuses on transitioning business-written code to a flexible IT-owned infrastructure.
We are looking for skilled R/Python developer who are comfortable on UNIX to assist in migrating business models already written in R from the GMETH windows platform to the FRP UNIX platform.

Responsibilities

Your responsibilities includes porting R code from Windows to UNIX, performing some changes to the R code to support methodology requests as needed, integrating the updated R code to our existing FRP framework (which is a set of microservices written in Python and Java), and assisting with the testing and deployment of the code
