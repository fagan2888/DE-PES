The Project

We are looking for a senior-level data engineer to create a data platform to provide tools for analysis, machine learning and API for many projects throughout the company. If you enjoy working with the latest technology from Amazon, and creating actionable info from large datasets, then this is the job for you.

What you'll do
  • Create a base set of tools to set up data pipelining jobs in a repeatable and scalable way.
  • Migrate existing data warehouses.
  • Help analysts and other data engineers to build business layer ETL and analysis - including machine learning tools and data aggregation for APIs.
  • Error detection, logging and monitoring.
Requisitos

Requirements
  • Well-versed user of Python and Spark.
  • Previous experience working with Amazon EMR, as well as PostgreSQL, Lambda, Step Functions and S3.
  • Excellent written and verbal communication skills.
  • Solid English proficiency to participate in project meet-ups (mandatory!).
  • Previous work loading data from third-party sources and REST APIs is a big plus!
