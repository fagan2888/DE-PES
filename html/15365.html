We are looking for Oracle Data Integrator Developer to work with Teradata projects.

You Will Be Responsible For
  • Requirements gathering.
  • Design, develop and implement the Data warehousing models using Teradata FSLDM.
  • Using Teradata FSLDM, address key areas in the data architecture – data integration and data analytics.
  • Use FSLDM for achieving an organized and integrated view of the business data.
  • Implementation of the Business Rules in the Logical Model.
  • Mapping the Source Data columns to the Destination Data columns.
  • Implementation of the Physical Model into the Database (Object Creation, PI, PPI Decisions, Compression Analysis, and Implementation).
Requirements:
  • Proficient in writing SQL’s to do the data transformation.
  • Possess strong knowledge on Oracle data integrator tool to do ETL development.
  • Proficient in Oracle BI tools to develop and deploy various BI related projects.
  • Application Development / Access of the Data.
  • To be fluent in English.
  • Strong interpersonal skills, positive attitude and ability to work with different teams.
Would be considered as a plus:
  • Knowledge in Teradata DW development and FS-LDM;
  • Banking/ BCBS 239 experience.
We offer:
  • Opportunity to be part of a rapidly expanding global organization with irreproachable reputation.
  • Opportunity to grow both professionally and personally.
  • Learning opportunities (Pluralsight, learning projects).
  • Inspiring working atmosphere and many engaging activities.
  • Clear career path and extra benefits.
  • Attractive compensation package: salary (3000 - 4400 EUR gross/ monthly) depending on competences you have.
