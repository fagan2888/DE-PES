A digital financial services house is on the hunt Data Scientist to join a fast paced and data-driven environment; developing advanced models using machine learning, deep learning and statistical methods!

In terms of calibre, we are looking for a strong, creative, detail-oriented, problem-solver; with the technical ability to implement solutions. The Hunt is on!

Requirements
  • 5+ years experience in an analytical/ engineering/ technical environment
  • Expert SQL skills required
  • Strong programming skills in Python or R
  • Hands-on analytical, feature creation and modelling experience
  • Experience in machine learning
  • Deep learning and big data technologies (such as Apache Spark) are a plus
  • Excellent communication skills
Duties And Responsibilities
  • Develop advanced models using machine learning, deep learning and statistical methods.
  • Demonstrate that models generalize to unseen data using cross-validation techniques.
  • Programme reproducible and testable code, to facilitate the deployment of models in a production environment.
Qualifications
  • A postgraduate degree (Honours, Masters or PhD) in Computer Science, Mathematics, Physics, Statistics, Engineering, or similar
