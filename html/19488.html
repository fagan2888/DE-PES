Job Role Summary

Primary functions: Development of scripts to standardize, automate, or support: data architecture, data quality management, reference and master data management, data integration, data governance, database development and design
Primary Deliverables: Scripts to standardize or automate: implementation of logical and physical data models, data profiling reports and data audits, data taxonomy and mapping, data sourcing, ETL services, database management, reporting
Activities: Process mapping, DevOps and Agile development, data design and modeling, data manipulation, scripting to standardize, automate, or support: data audit, definition of hierarchies and affiliations, data quality & cleansing, ETL & integration

Job Role Responsibilities

  • Demonstrate critical thinking, analytical skills, and employ judgment to offer thoughtful, concise input toward resolutions of problems
  • Be able to translate data requirements into business processes and reverse engineer business processes into data requirements and automated tools and scripts
  • Competency in TOAD for Oracle, SQL Server Database, programming in Azure Cloud Platform and enterprise reporting tools (e.g., Tableau)
  • Advanced competency in microservice architecture and functional programming in modern languages
  • Comprehension of DevOps and Agile development and application to data centric architecture and solutions
  • Basic Project Management (agile & waterfall) and Business Process modeling
  • Strong communication and interpersonal skills with strong English proficiency
  • Leadership skills needed to successfully promote ideas, coordinate work activities, and plan deliverables within a project team.
  • Minimum Bachelor’s Degree in Engineering Technology, Computer Science, or a related field with equivalent experience

    • Expected Level of Proficiency

  • Experience working with Azure Functions, Logic Apps, API management, Data Factory
  • Advanced expert database handling languages (SQL and variants)
  • Practical skill sets in BI and Data visualization tools
  • Knowledge of other programming languages and ability to write code (e.g., Java, .net, python, XML)
  • Expert in data manipulation tools such as: SSMS, SSIS, TOAD, Access, Excel, Info Steward, Data Services
  • Data Modeling using tools like Erwin/ERStudio is a plus
  • Experience working with Snowflake Data Warehouse on Azure Cloud will be a plus
  • Working knowledge of SAP is a plus
  • Fluent in english
