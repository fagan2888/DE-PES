Test Automation Engineer

Project Information
  • Industry: Finance / Banking
  • Type of project: System development / maintenance
  • Location: Gdynia
  • Business trips: Occasional trips to Denmark/Finland/Sweden
  • Project language: English (required), Polish
  • Remuneration: 100 - 125 PLN/H
  • Project length: 3 months+extension


Responsibilities
  • Analyze and comment requirements and technical designs from a test perspective
  • Designing, implementing and executing test cases by automated test methods
  • Contribute in defining the overall test plan
  • Work actively with the whole team to secure high quality of the delivery within expected timelines
  • Gathering and managing test data
  • Monitoring and documenting test results
  • Report, debug and verify defects
  • Creatively removing impediments
  • Recommending appropriate actions for test results


Requirements
  • DB/SQL; HP Software - ALM/QC; Software - GIT; Test Automation; XML/JSON/FILES
  • Minimum 2 years of testing experience preferably in banking reporting or accounting domain and /or data warehousing projects
  • Exposure in life cycle testing of Savings and Lending products (Deposits, Accounts and Loans)
  • Understand test data dependencies for a given context
  • Executing and reporting tests and defects for complex software systems
  • Analytical and problem solving skills
  • Knowledge and experience with SQL databases, Hadoop (HIVE, HDFS), message queues, text files (JSON, XML, CSV, plan text), MS Office Excel, Data Warehousing concepts, Teradata; for test automation object oriented programming experience – preferably C#, nunit, Web services, reporting in Report Portal
  • Knowledge and experience with tools HP ALM test management tool, Atlassian suite (Jira, Confluence); for test automation GIT version control system and Unix-like systems, experience with Continuous Integration environment
  • Experience of working with Agile methodology will be highly valued

  • Challenging international projects in a Scandinavian business culture
  • Transparently built relations based on trust and fair play
  • Benefits: Medicover, MultiSport
  • Relocation package
