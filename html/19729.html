As a Software Testing Engineer you will be responsible for:
  • Developing automation code (Java or Groovy) to build test frameworks, improve functionality and automate testing infrastructures
  • Overseeing the scope of releases changes and owning the responsibility of release testing
  • Proactively bringing innovation from the marketplace, contributing to the wider community and striving for innovation
  • Being a collaborative and proactive member of the teams in Krakow, US and other global teams
Do what you love

To Be Successful In This Role You Will
  • Have experience developing code in Java or Groovy
  • Have experience in Automation Testing, Web API Testing and familiarity with TestNG and/or JUnit frameworks
  • Demonstrate comprehensive understanding of the internet, including TCP/IP, DNS, HTTP, HTML, and SSL protocols
  • Demonstrate working knowledge of automation frameworks and deploy/build tools
  • Familiarity with Agile methodologies and best practices
