Job Summary

We are now looking for a Data Analytics Developer to support production technology evolution and efficiency improvement from advanced data analysis perspective.

Responsibilities
  • You will be responsible for product lifecycle data analytics
  • Test time reduction use Machine Learning algorithm. Module base calibration, Conditional-ruled testing ,HW Database analysis, tester Maintenance analysis, etc.
  • Site claim data analysis
  • Repair data link to supplier key component data
  • Interface to smart factory
  • DFM input

Key Qualifications
  • MS in Computer Science, Mathematics, statistics, Artificial Intelligence, or related
  • 2 years of software development in Python / C++ / Java
  • Knowledge on in bigdata like Hadoop ecosystem, HDFS, Spark, etc.
  • Hands-on experience with one or more ML algorithms such as Regression, Clustering, Trees/Random Forest, Bayesian statistics, SVM, Neural networks, deep learning/reinforcement learning, etc.
  • Knowledge in statistics, analytics methodologies and related communication and presentation skills
  • Fluent in English speaking/writing
