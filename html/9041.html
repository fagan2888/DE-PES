Are you a visionary creative thinker, with an intuitive customer-centric approach? Do you want to make a difference in our company and be part of creating something new?

Are you able to take the lead, looking after challenging timelines and working autonomously? If the answer is yes, then we would like to hear from you.

We are looking for Senior AWS CLoud Engineer with the following details

The Team

The Cloud operations team encompass the process of managing and delivering cloud services and infrastructure to either an internal or an external user base. This involves ensuring peak performance and maintaining availability in order to satisfy the needs and expectations of customers and meet service level agreement standards. Team is operating in follow-the-sun (24 x 7) model around the globe (Europe, Asia, Latin America), to ensure the reliability of PMI cloud services across all product lines in tight coordination with Cybersecurity and Solution and Infrastructure Architect teams.

We have implemented Agile, Scrum and DevOps methodologies, we are also continuing to apply Agile Change Management. So far we had completed the integration with our own DevOps tool and ticketing application, moreover training has been conducted and some pilots have been implemented.

The Role

Working as part of a distributed team you will help to deploy, configure and manage public cloud infrastructure at PMI.

As part of the Cloud Operations team you will provide break and fix to ensure the infrastructure on top of cloud services is up and running, working together with application support team to deploy and troubleshoot infrastructure related issue as well as participating in architecture/technology review, moreover security compliant is a must which need to be remediated to align with PMI security standard.

You will be part of growing team who responsible for leading the deployment and adoption Cloud based technologies, AWS Amazon and VMWare. Play an active role in helping customers to plan and execute cloud migrations, implementations as well as troubleshooting cloud infrastructure related issue.

This is very exciting role, you will be working all day with the most exciting Public Cloud Technology, Amazon AWS, and Private Cloud Technology, VMWare-based. Involve in the Cloud Projects helping customers/business to build and migrate from traditional infrastructure to cloud-based technologies. Learn from team members with over 10 years’ experience in IT within worldwide enterprise technology landscapes. Continuously learning new Cloud Technologies (AWS Amazon and VMWare based)

Mostly we’re using cloud-based technologies, such as AWS services (EC2, autoscaling, cloudformation, ELB, Route53, EFS, SES, S3, Lambda), VMWare (vRO, vRA, vROps, vSphere), scripting (Javascript, Python, Powershell, Json, Yaml, linux bash), Operating system, Windows based server and Linux.

Responsibilities

• Design, Implement and Manage PMI public Cloud Infrastructure based on AWS solution
• Provide break/fix support for the cloud environment
• Develop and execute test cases for cloud environment
• Actively participate to review and design Architecture and find the best solution based on the requirements
• Remediate all the security issues based on PMI security standard
• Design, implement and manage IS processes for the cloud environment support
• Develop and propose continuous improvement, waste busting and implement it

Skills & experience

• An IT professional with at least 5 years’ experience in multinational enterprise environments
• Experienced in hands on Amazon Web Service (AWS)
• Able to Scripting (Python or PowerShell, etc.) is a MUST
• Collaborative and open working style
• University degree
• Fluent English (spoken and written)
