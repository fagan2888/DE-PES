Position Description

Vil du være med til at implementere fremtidens analytiske løsninger for danske og internationale virksomheder og have en nøglerolle i en global vækstvirksomhed?

CGI er verdens 5. største it-servicevirksomhed og lever for at levere fremtidens IT-løsninger til vores kunder. Vi søger en ambitiøs Data Engineer, der vil være med til at levere komplekse løsninger en bred vifte af kunder i en global kontekst.

Du vil blive en del af CGI NEXT, der arbejder med de nyeste teknologier inden for cloud computing, datawarehouse, Business Intelligence og kunstig intelligence. Du vil blive en del af CGI´s team af specialister, når det kommer til værdiskabelse med data.

Din profil

Vi Leder Efter Kandidater Med Fortrukne Erfaringer

  • Kandidatgrad inden for datalogi, software eller andet relevant teknisk felt
  • Har solid erfaring med data modellering (Data vault og Dimensional modellering)
  • Har praktisk erfaring med data migrering fra ’on-premise’ til cloud løsninger
  • Har solid erfaring med SQL og RDBMS
  • Har solidt erfaring med Microsoft Business Intelligence Stack, samt Azure ressourcer som f.eks. Azure SQL Database, Azure SQL Data Warehouse, Azure Data Lake, Azure Data Factory
  • Har solid erfaring med at udvikling af data-flows og -transformationer
  • Har erfaring med DevOps processor, som CI/CD, automatiske test og Infrastructure as Code
  • Har forståelse for Master Data Management (MDM) og Data Quality værktøjer og processor
  • Arbejdet med Power BI eller andre rapporteringsværktøjer

    • Et Plus Vil Være Kendskab/erfaring Med

  • Optimering af data forespørgsler, analyse af data problemer, vedligeholdelse af dataintegritet samt generering af automatiseret rapportering
  • Forståelse for .NET, C#, Python og/eller R
  • Produktionssætning af AI / Machine learning løsninger
  • Containerization, Azure IaaS og Azure SaaS løsninger
  • Udvikling af data service API’er, der eksponerer data til både BI and Machine Learning løsninger

    • Det tilbyder vi hos CGI
  • Branchens bedste brand og personlig udvikling med de muligheder, der findes i et internationalt miljø
  • Medejerskab af verdens 5. største it-servicevirksomhed gennem vores Share Purchase Program
  • Attraktiv pensionsordning, sundhedsforsikring, kantineforhold og en initiativrig aktivitetsforening
  • Et godt socialt miljø med stor respekt for hinandens faglige og personlige kompetencer
  • Deltagelse i internt ERFA-samarbejde

    • Som ansat hos CGI tilbyder vi et spændende og selvstændigt job i en professionel virksomhed omgivet af kompetente kolleger, høj faglighed og professionalisme. En førende virksomhed med stabil vækst og en sund økonomi. I CGI taler vi ikke kun om at skabe balance mellem arbejdsliv og privatliv – vi udøver det i praksis. Deraf stor fleksibilitet i forhold til planlægning af dit arbejdsliv.

    Er du den, vi søger?
    Vi glæder os til at høre fra dig hurtigst muligt. Stillingen ønskes besat snarest.
