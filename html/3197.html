Position Description Overview

  • Design and implementation of IT solutions based on the presented analytical and technical documentation;
  • Creating technical documentation;
  • Reviewing analytical documentation;
  • Technical support for existing systems;
  • Reporting work progress to superiors.
  • Very good knowledge of Java 8+ and object-oriented programming issues;
  • Experience in independent design, implementation, documentation and testing of application components;
  • Practical knowledge of JEE 8, Spring, and Hibernate technologies;
  • Practical knowledge of tools supporting the software development process: Maven, Git, Gerrit, Jenkins, Jira;
  • Knowledge of relational databases, in particular Oracle;
  • Knowledge of creating a SPA application (GWT, AngularJs technology).
  • Knowledge of HTML5, CSS3
  • Experience in systems integration (SOAP, REST web services)
  • Experience in developing eCommerce solutions is welcome
  • Responsibility and diligence, readiness to create new solutions in line with the adopted process;
  • Ability to solve problems independently, ease of learning new technologies;
  • Ability to work effectively in a multi-person team;
  • Min. Required 3-year work experience in a similar position.
