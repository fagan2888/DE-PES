Job Description

We seek a self-motivated, experienced, and detail-oriented software engineer to join our team and help develop new frameworks for multi-sensor autonomous localization in harsh environments.

Support a multi-disciplinary team of research scientists and engineers with backgrounds in neuroscience, aerospace engineering, computer engineering, computer science, and mechanical engineering. We work across a myriad of domains including autonomous robotics, supervised and unsupervised deep learning, machine learning, artificial intelligence, neuromorphic sensing, and human-machine teaming.

Responsibilities

Spearhead development of a novel software architecture for multi-sensory localization across the entire software development life-cycle (SDLC)
Integrate disparate algorithms and open-source software components into holistic, production-level software packages
Assist in writing use cases and software specifications
Develop automated-/unit-testing routines
Document software using industry-standard best practices.
Manage and maintain internal Git repositories

Required Qualifications

Strong knowledge of C, C++, and Python

Bachelor's or master's degree in electrical engineering, computer science, software engineering, computer engineering, or a related discipline

3+ years' experience in Linux-based software development

Extensive experience with Git and managing repositories, workflows, and submodules

Experience with all stages of the software development life cycle (SDLC)

Familiar with Agile software development practices and working in an Agile environment

Strong mathematical skills

U.S. Citizenship is required

Must be eligible for a SECRET clearance

Desired Qualifications

2+ years' experience in robotics/autonomous systems Experience with the Robot Operating System (ROS) Hardware-based software development and testing experience with 3D Computer Graphics and Computer Vision Experience with graphical user interface (GUI) programming (Qt) Experience with Android programming
