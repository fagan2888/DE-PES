Data Engineer: Cliente en Banca:
: Tener de 3 a más años de experiencia en gestión de datos que involucre diseño, construcción y mantenimiento de modelos de datos operacionales y analíticos.
: Bachiller o estudios en computer science, Ing. Sistemas o campos relacionados.
: Experiencia en cloud: google o azure, no hay una preferencia
: Ingeniero programador en python y R (tener capacidad de programacion es indispensable)
: Experiencia con herramientas de Inteligencia de Negocios (ej: Tableau, Power BI, etc)
: Experiencia trabajando con bases de datos relacionales (Ej: SQL Server, DB2)
: Habilidades de comunicación para el entendimiento de las necesidades de los equipos y una correcta captura de requerimientos.
: Habilidades de análisis para traducir los requerimientos del negocio en modelos de datos conceptuales, lógico y físicos.
: Nivel de inglés: basico o intermedio
: Experiencia con flujos de procesamiento de datos en tiempo real o ETL
: Deseable experiencia con herramientas de Inteligencia de Negocios (ej: Tableau, Power BI, etc)

Funciones o Temas Que Verá

: Procesos de datos en la nube o en plataforma hacia la nube.

: Va a hacer analisis estadísticobasico sobre la data.
: Trabajaría en la integración de las tecnologias de nubes con las tecnologias de bases de datos relacionales y no relacionales.
