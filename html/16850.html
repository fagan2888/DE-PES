Embedded Software Engineer

Aerotek Richmond, CA

1 day ago

Apply on company website


This individual will work on middleware that will interface with the company's products and Linux on northbound end and will interface with API's all within application layer. They will develop API's so customers can connect their products with the company's products. On a typical day, developers will have to look at the framework, and they'll be expected to know technologies underneath it. Mostly modification of framework since there's already established framework. There will be debugging and and issue resolution.

MUST HAVES
1+ year real-time operating systems experience
1+ year applications layer experience


NICE TO HAVES:
* Knowledge of how to apply the patch to the linux kernel (point 1 below, the rest are nice to haves)
* Embedded Linux
* Developing under desktop Linux
* Git, repo, bitbake, and yocto
* Designing application programming interfaces (API's)
* Continuous integration
* IP networking
* IoT, security, and cryptography
* Open hardware platforms (Arduino, Raspberry PI, MangOH, etc.)
* Previous work on open source projects
* A love of writing documentation
* Willing to give talks in front of groups of people
* Willing to write whitepapers
* Device drivers
RTOS experience
* PAN/HAN
