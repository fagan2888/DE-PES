Data Engineer
  • Large Enterprise Data Warehouse Program
  • Surry Hills Location
  • 12 Month Contract


Be part of an exciting new program which will involve:

  • Implementing a new Enterprise Data Warehouse (EDW) and Analytics Platform - Delivering a variety of data services that range from web services to analytical datasets,
  • Building capabilities to identify and treat data quality issues
  • Integrating our new testing and devops capabilities.
  • Leveraging the automation assets that we have delivered so we can utilize the value of these to our organization
  • Enabling a fail fast methodology across our Branch to shorten the feedback loop and increase the delivery cadance

    • The Ideal Candidate Will Possess

    Technical Experience Experience with any of the following technologies:

  • Experience in ETL/Data warehousing
  • Experience in other forms of development or full stack - SAS, Teradata, Python, RStudio
  • VMWare, Linux and Windows
  • Webservers: NGINX
  • Atlassian suite (on-prem): JIRA, Bamboo, Bitbucket, Confluence, Crucible, Fisheye
  • Frameworks: Django, React, Selenium, NodeJS
  • Configuration Management: Ansible
  • Testing Tools: XRAY, Fitiness, DBFIT, jUnit, ngUnit, Selenium
