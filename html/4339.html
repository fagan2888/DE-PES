О компании

Компания Align Technology Inc - один из мировых лидеров в разработке инновационных медицинских технологий. Компания производит уникальный продукт - Invisalign, который дает возможность людям приобрести идеальную улыбку простым и эстетичным способом (без традиционных брекетов). Align Technology - американская корпорация, штаб квартира расположена в Кремниевой долина, Калифорнии.

Московский филиал Align Technology, Inc. – это интенсивно растущее подразделение R&D, работающее в области digital orthodontics.

Основные задачи команды:

Команда Treatment Planning Core отвечает за подготовку данных, взаимодействие с командой разметки, планирование, создание и обучение ML моделей для 3D сегментации а также за использование предсказаний этих моделей в приложениях.

Как делаем:
  • отбираем данные из базы с помощью SQL запросов
  • скачиваем данные с помощью python скриптов
  • подготавливаем данные для разметки ground truth для модели
  • разрабатываем необходимую функциональность на C++ в CAD для выделения нужных данных из цифровых сканов пациентов и/или лечений
  • подготавливаем отчеты и строим графики в jupyter (pyplot, matplotlib)
  • используем распределенную систему для обработки больших объемов данных
  • сопровождаем процесс обучения модели, регулярно анализируем и выделяем главные проблемы модели
  • планируем и разрабатываем архитектуру ML решения в Amazon Web Services
  • разрабатываем протокол и интегрируем подготовленную модель с весами в web service
  • подготавливаем docker container для работы в AWS
Какие технологии используем:
  • C++ 11
  • SQL для отбора данных из баз
  • работа с Amazon S3
  • Python

Что вам нужно будет делать в проекте:

Подготавливать данные для обучения нейронной сети и автоматизировать этот процесс. Настраивать ETL пакеты для подготовки данных. Взаимодействовать с командой разметки а также с другими командами. Определять медленные места в процессе подготовки данных и ускорять их. Оценивать, проверять и использовать последние инструменты в big data.

От разработчика нужно:
  • Базовые навыки программирования на C++11, python
  • Базовое знание SQL
  • Знания теории вероятностей и математической статистики
  • Возможность уделять стажировке минимум 20 часов в неделю
  • Возможность работать после стажировки полный день
  • Готовность быстро самостоятельно учиться

Что будет плюсом:
  • English от уровня intermediate
  • Знакомство с Amazon Web Services
  • знакомство с jupyter
Что мы предлагаем:
  • Оплачиваемая стажировка с гибким графиком, от 20 часов в неделю
  • Участие в реальных R&D проектах
  • Оплата профильных курсов, тренингов
  • Возможность трудоустройства в компании по результатам стажировки
