Cloud Applications Consultant (Job Code: 1900002917)

Wipro Limited Yokohama, JP

1日前

Job Description

Cloud Application Consultant. The role will involve front ending cloud solutions to customers in the application space and require complete hands-on approach. The consultant would be expected to design the right application solution architectures for Microservices, Containers, Dev-Ops using Cloud functionalities like PAAS, etc. It will also involve a decent amount of proposal making, carrying out Proof of Concepts and preparing RFP responses. Will have to work with various teams across the globe to provide the consulting support to the customers in Japan. The consultant is expected to understand the existing landscape of the customer and consult on the Cloud adoption possibilities leading with experience.

Skills
  • Should have good hands-on cloud-ready application architecture experience ( 3+ years ) with good working knowledge of minimum 1 CSP ( AWS / Azure / GCP ).
  • 8+ years of experience on Application development ( Java, DotNet, HTML5, etc. ), CI/CD workflows and application refactorization / rearchitecture will also be expected.
  • Looking for application developer who has used APIs to invoke call to Cloud services.
  • Experience developing applications which relies on leveraging Cloud Services.
  • Extensive understanding of one or more programming languages .
  • Proven ability to effectively communicate architectural standards and best practices for scalable solutions.
  • Good understanding of different problem-solving strategies.
  • Good understanding of serverless & container design concepts and patterns
  • 3+ years of experience using CI/CD tools (git, Jenkins, Bitbucket, AWS Code Pipeline, Nexus, Artifactory, Terraform, CloudFormation)
  • 3 years of experience with modern automation, DevOps, and Agile development
  • Experience with HTTP APIs and RESTful web services is a plus
  • AWS Certifications are a Plus
