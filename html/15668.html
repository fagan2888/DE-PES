Data Engineer
DCS are currently looking for Data Engineers to work in Cardiff for a
National Client.
Requirement: Data Engineers x6
Location: Cardiff
Start Date: Monday 31/10/16
Duration: 6 weeks
Working Hrs: Monday – Friday 08:30 – 16:30
Day Rate: £110

For This Role, You’ll Need The Following
  • CSCS Card
  • All hand tools and terminating tools (110 punch down, Krone Tool, ModTap
tester
  • Full 5 Point PPE (safety boots, yellow high vis jacket, hard hat, goggles &
gloves

Job Description
  • Installing CAT6 Data Cabling
  • Terminating CAT6 Patch Panels
  • Terminating CAT6 Floor Boxes and CAT6 GOP Boxes or Wall Outlets
  • Testing CAT6 Structured Cabling using a Fluke tester.
  • Labelling data cables, patch panels and floor outlets
  • Carrying out any tasks as requested by our client on site.
Key Words Would Include

Data Cabling / CAT6 / CAT5E / CAT5 / / Data Installation / Terminating /
Testing / Fibre Optic / Fibre Splicing / Network Cabling / Moves, Adds &
Changes /
