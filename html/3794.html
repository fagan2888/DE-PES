ESSENTIAL DUTIES AND RESPONSIBILITIES:


  • Analyze electromechanical systems requirements and decompose into logical segments in preparation of test cases design.Manage resolution of architecture issues. Enforce design commonality (as practical) across concepts in close cooperation with Systems Engineer.
  • Develop complete system architecture in coordination with the HW, SW and ME design teams.
  • Create and maintain a comprehensive test strategy for each project, in accordance with ASPICE.
  • Create and maintain system functional test cases using Gentherm tools based on the assigned System Level requirements. Maintain test cases according to changing requirements
  • Support assigned project software level test automation activities using CAN tools and support in-vehicle & breadboard activities when required.
  • Develop system level integration test plans, lead execution of manual/automated test cases using Gentherm tools.
  • Prepare technical documentation, using IBM JAZZ reporting tools (RPE) to generate test reports as needed for every assigned software release upon completion of test cases execution.


YOUR PROFILE:


  • Bachelor’s Degree in Electrical Engineering, Computer Science or relatedWorking knowledge of automotive specifications for electromechanical systems
  • 3+ years of experience in embedded software development or test
  • 3+ years of experience in C/C++ or other structured language
  • Experience developing and debugging software in real-time, embedded environment
  • Experience with test automation systems (eg.: Vector VT System)
  • Creative problem-solver capable of creating and reproducing complex software defects
  • Excellent communication skills – verbal and written in English
