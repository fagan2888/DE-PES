Seargin is looking for a Senior Data Engineer

Position : Senior Data Engineer

Technologies : DevOps, Python, Data Engineering

Localization : Cracow/Warsaw/Wroclaw

Area : R&D

Form of employment : Permanent / B2B

Experience Level : Senior

Main Tasks For a Senior Data Engineer Will Be

Be responsible for the advancement of next generation cloud-native data hub with built-in Artificial Intelligence Capabilities;

Gathering data, consolidating, processing, visualizing and applying machine learning models to generate business insights and value.

The Candidate Should Have

Great expertise in using Java or Python ;

Hands-on experience in DevOps and IaC tooling;

Interest in different languages and new technologies;

Knowledge of distributed systems design problems;

Understanding of distributed data storage systems/formats using parallel processes and/or columnar data stores such as AWS Redshift, Elastic Search;

SOLID, Clean Code.

Flexibility, superb analytical and problem solving skills;

Excellent communication and interpersonal skills;

Team-oriented, detail-oriented, efficient and solution-oriented attitude;

Ability to work independently and in a team;

Very good English skills (written and spoken).

It Would Be a Plus If The Candidate Had

Expertise in online and offline data processing tools and technologies (Storm, Kafka, Spark, Flink);

Understanding of Cloud Computing on multiple Amazon Web Services or any other public cloud offering;

Experience with Data Collection + Ingestion, Data Visualization, ETL, Data Processing Pipeline and Data Modelling;

Acknowledgment of Continuous Integration/Continuous Deployment Environment.

The Candidate Can Expect

Permanent / B2B Contract;

Professional advancement in the fields of Data Science, Big Data and Machine Learning;

Skill improvement thanks to the experienced team of international software engineers;

Tasks on bleeding edge public cloud architecture and data engineering framework;

A chance to bring in your own ideas and thoughts; be a part of the decision making process;

A challenging job in an international and multilingual environment;

Attractive, competitive compensation;

High independence, pleasant work culture, modern office space, DevOps methodology;

Access to chill out zones, tech guilds, lightning talks, multisport card, life and health insurance, companies Academy and programs, bike parking space;

Flexible work schedule;

Training courses and conference participation.
