Halian is looking for an “Oracle Database Engineer” required in order to work for an EU Institution in Luxembourg. Ideally with 5 years of experience in a similar position and environment described below.

About The Role
  • Main Tasks:
  • Installation, configuration, upgrade and patching of Oracle databases and applications
  • Oracle tablespace management
  • Management of network database connections, access control and logging
  • Backup and restore operations
  • Problem identification and analysis, incident resolution
  • Integration of standard/custom applications in the production environment
  • Management of Very Large Databases (partitioning and tuning)
  • Management of Databases replication mechanisms (ASM, DataGuard)
  • Expected results:
    • Database availability
    • Database monitoring and adequate performance
    • Database integrity and security
    • Assistance to the local IT team on advanced database architectures and technologies
    • Assistance and expertise for corrective and evolutive maintenance of the production systems


    Key Skills & Background
    • Technical experience:
    • Oracle Database Enterprise Edition 10gR2 to 11gR2
    • Oracle/SUN Solaris 10 or 11 basic administration
    • Oracle RMAN, EMC2 Legato Networker client
    • Oracle NET Services
    • Oracle Financials 11i
    • Oracle ASM
  • Can be considered as asset:
    • Web related technologies, XML
    • Open Source standards and software
    • Oracle RAC
    • Oracle WebLogic Suite 11G
    • Oracle Enterprise Manager
    • PostgreSQL
    • MySQL
    • Red Hat Enterprise Linux 6 advanced administration
  • Certifications:
    • Oracle Database Administrator Certified Professional
  • Languages:
    • English
    • French

    Details
    • Location: Luxembourg

    • Start Date: ASAP
