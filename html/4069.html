Data Engineer

Techno Wise (INDIA) Ahmedabad, IN

2 weeks ago

Apply on company website
Client of Techno Wise

We are looking for data engineers (preferably with ETL background):
  • Designing data schemas/stores and data pipelines for storing and processing of different kind of structured and unstructured data sets like transaction data, review data, video feeds, process data, email data, social feed data and many more
  • Executing batch jobs on our custom-built computing cluster or any standard ETL tools or using custom code in SQL or Java or Python
  • Working closely with the data science team to integrate models to tag data to our data store attached to a KPI or raw data
  • Working closely with our backend engineering team to build a robust suite of libraries for extracting and manipulating data for our apps
  • Add new IoT data connectors to byte prophecy core platform by researching on third party protocols and standards
  • Create libraries for data quality assurance or data sanity checks
Required Skills
  • Experience with both SQL (MySQL) and Columnar (MariaDB/InfiniDB), NoSQL (Cassandra) and big data (Hadoop) data stores
  • Familiarity with programming best practices, design patterns, version control systems
  • A sound understanding of parallel/distributed programming
  • Attitude to go extra mile for the completion of the assigned work
  • Good command in Python or Scala or JavaScript
  • The ability to work effectively with people from a variety of backgrounds
