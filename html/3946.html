Pau, France

Cette société de conseil spécialisée dans le domaine du Big Data collabore sur de nombreux projets avec des grands groupes de l'industrie Française.

Aujourd'hui, elle travaille sur un projet lié au Big Data avec un acteur majeur de l'aéronautique, cette mission est basée à Pau .

En tant que DATA ENGINEER vous serez amené à traiter un très grand nombre de données, intéressé(e) par les problématiques autour de la mise en place d'outils Machine Learning ? Cette opportunité est faite pour vous !

Le Poste

Au sein de l'équipe technique vous travaillerez en collaboration avec les Data Scientist :
  • Vous collecterez et stockerez les données
  • Vous optimiserez les traitements
  • Vous rédigerez un code de qualité
  • Vous utiliserez des technologies comme Kafka, MongoDB ou SQL pour répondre à ces challenges
  • Vous serez le point de référence des Data Scientits concernant l'environnement de travail proposé.

Le Profil
  • Vous êtes diplômé(e) d'une Grande Ecole d'ingénieur ou avez une forte expérience en big data
  • Vous maitrisez Java et prêt à découvrir de nouveaux langages
  • Vous avez de connaissances en Spark et Hadoop
  • Vous menez une démarche scientifique et avez un bon relationnel pour présenter vos travaux et communiquer avec les équipes.
  • Vous aimez partager vos connaissances et vous voulez monter en compétences sur les sujets data
  • Vous êtes force de proposition et pouvez travailler de façon autonome

Comment Postuler

Si vous êtes intéressé(e), merci de faire part de votre CV à Pierre Gerbeau
