Apply on company website

Role description

We are currently looking for BI Data Engineer to fill a permanent position in the BI & Reporting team. The successful candidate will work in a team managing data in a data warehouse solution and make it possible for the organisation to get the most out of the data in terms of reporting, statistics and form the basis of the decision making and analytical processes.

In this role, you will be part of the BI & Reporting team responsible for managing reporting solutions and tools in the data warehouse environment. It involves setting up and maintaining data aggregates, scheduled jobs, building and managing APEX, Qlikview/Qliksense reports, setting up new streaming jobs on Spark, and new tools and technologies used to fetch data from the data warehouse for campaigns & marketing activities, analysis and customer communication. It is required to interact with other departments to gather requirements and implement and support solutions.

Key accountabilities of this role are:

  • Advanced Oracle SQL writing and optimisation queries over large data sets
  • Creating and maintaining PL/SQL packages and Database objects
  • Oracle APEX development
  • Advanced Qlikview/Qliksense Scripting
  • Data investigations and problem-solving
  • Setting up Apache Spark streaming jobs
  • Performing tuning queries
  • Work and enhance testing framework using Python & Java
  • Writing shell scripts for automation of file transfers
  • Setting up and managing Qlik dashboards and reports
  • Ad hoc SQL reporting
  • Some reporting experience preferably Spark & Spark SQL
  • Stakeholder engagement with internal and 3rd party users
  • Working with teams across geographical locations
  • Supporting Data Platforms (including occasional on-call support & incident management)

What have you done?

  • Experience with dealing with large data volumes
  • Strong background of working with Oracle databases (11g and above) in a Developer/Analyst role
  • Advanced scripting in Oracle SQL and PL/SQL
  • Understanding of Data Warehouse concepts
  • Experience with data streaming solutions
  • Experience with setting up dashboards and reports for business end users
  • Advanced knowledge in Unix Shell
  • Working knowledge of Python or core Java
  • Familiar with version control (GIT, SVN)
  • Previously worked in an Agile/scrum framework
  • Passionate for excellence in his/her work
  • Passionate about complex data structures and problem-solving
  • Logical and inquisitive able to pick up new data tools/concepts quickly
  • Highly organised, detailed-oriented and thoroughly able to work independently and as part of a team
  • Previous experience in the gaming/gambling sector would be beneficial
  • Native level English language skills
