Job Description


• Understand and analyse the business requirements meticulously.
• Processing, cleansing, and verifying the integrity of data used for analysis.
• Apply effective Data Mining techniques to perform an extensive EDA and gather insights.
• Apply the appropriate Statistical techniques and/or Machine Learning (ML) on both structured and unstructured data.
• Write clean and clear codes in a modular fashion in Python .
• Identify unconventional data sources that can be used for predictive models, identify ways that data can be captured and then use that data to further refine existing models.
• Develop new analytical methods/tools as per the requirement.
• Exposure to popular frameworks like Tensorflow, Pytorch etc.,
• Demonstrate effective written and verbal communication skills.

Good to have Big data technologies.


Mandatory :

Atleast 3 years hands on experience in python

Must have worked on NLP and AI

Can join in max 30 days
